### Sento/Minamaze

Post/Page and Archive can have different layouts.
Uses [Redux Framework](http://reduxframework.com/)

```
| Pre Header menu               |
| Header            Header menu |
| ----------------------------- |
|            Slider             |
|        Message/Button         | (optional)
|        Featured Area          | (optional, link to custom pages)
| ----------------------------- |
|      Post/Page      | Primary |
|      Content        | Widget  |
|                     | Area    |
| ----------------------------- |
| Footer Sidebar                | (optional, multiple columns)
```

### Educate
