## CLI

[About this documentation · GitBook Toolchain Documentation](https://toolchain.gitbook.com/)


```sh
npm install -g gitbook-cli
gitbook ls
gitboot ls-remote

# create {SUMMARY,README}.{md,adoc}
gitbook init
gitbook install
gitbook serve --port 4321

gitbook help
gitbook build
# convert with ebook-convert
gitbook pdf
gitbook epub
gitbook mobi
```
